# Tetris
[![Python 3.x](https://img.shields.io/badge/python-3.x-blue.svg)](https://www.python.org/downloads/)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgray.svg)](https://github.com/)
[![Build Status](https://github.com/neohiro/Tetris/actions/workflows/release.yml/badge.svg)](https://github.com/neohiro/Tetris/actions)

A small retro Tetris game for Windows, Linux & macOS, written in Python using Pygame.

## Features
- Classic Tetris gameplay with modern, smooth mechanics.
- Neon retro aesthetics with glowing effects.
- Dynamic sound generation.
- Fullscreen support.
- Local high score saving.

## Installation
You can download the compiled standalone executable for your operating system from the Releases tab.

If you prefer to run from source, install the requirements:
```bash
pip install pygame numpy
python Tetris.PY
```
